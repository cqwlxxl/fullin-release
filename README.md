# fullin-release

#### 介绍

浮音播放器(fullin)发布页

一款支持桌面精准歌词的音乐播放器，桌面歌词流畅且能被OBS直播软件获取，支持krc与lrc歌词

使用本软件前，请先安装LAVFilters解码器

#### 版本更新
