# fullin-release

#### 介紹

浮音播放器(fullin)發布頁

一款支持桌面精準歌詞的音樂播放器，桌面歌詞流暢且能被OBS直播軟件獲取，支持krc與lrc歌詞

使用本軟件前，請先安裝LAVFilters解碼器

#### 版本更新
